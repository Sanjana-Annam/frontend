# WEEP — Women’s Entrepreneurship Empowerment Program


A starter website for the WEEP initiative built with React + Vite + Tailwind CSS. It demonstrates the key product pages, program listings, mentor sections, testimonials, and a mock registration flow.


## Quick start
1. Save the files from this repository structure.
2. Run `npm install`.
3. Run `npm run dev` to start the dev server.


## How to extend
- Replace `src/services/api.js` with real API calls.
- Add authentication for mentors and sellers.
- Build admin dashboard to approve applicants and list products.
- Integrate payment gateway for marketplace.
